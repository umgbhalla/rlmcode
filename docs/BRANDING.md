# rlmcode — brand

## Name

`rlmcode` — always lowercase, one word, monospace. Never "RLMCode",
"RLM Code", or "Rlmcode". In prose, set it in `code` style when the
medium allows.

## Tagline

> A self-orchestrating TUI coding agent.

## One-line pitch

The model doesn't just call tools — mid-turn it authors and runs a JS
orchestration script, and a live nested node-tree renders every branch as
it happens.

## Aesthetic

Stark black & white. Terminal-minimal.

- **Color:** none. Black background (`#000`), white/grey text. No accent
  hue, no gradients, no shadows.
- **Type:** monospace only (`ui-monospace`, `SF Mono`, `Menlo`,
  `Monaco`, `Consolas`, monospace).
- **Shape:** no rounded corners, no cards, no marketing fluff. Square
  edges or none at all. Generous whitespace.
- **Texture:** flat. It should read like a terminal, not a SaaS page.

## Voice

Terse, technical, caveman-direct. State what it does; drop the adjectives.

- Say "the model authors + runs a JS orchestration script." Don't say
  "revolutionary AI-powered orchestration platform."
- No exclamation marks, no hype, no emoji.
- Prefer verbs and nouns over claims. Numbers only if true.
